About The Application of This System

This application is an implementation of a Deep Learning model for Breast Cancer classification.

Model

Using InceptionResNetV2 with CBAM (Convolutional Block Attention Module) to improve focus on important features.

Preprocessing

Each image is processed with CLAHE (Contrast Limited Adaptive Histogram Equalization) to sharpen image details.

Backend

Built using Flask (Python) to serve the model and integrated with the Google Gemini LLM to provide dynamic explanations.

Frontend

Clean and responsive interface (UI/UX) design built using Bootstrap 5 and external CSS/JS files.
