Tentang Proyek Ini

Aplikasi ini adalah implementasi dari model Deep Learning untuk klasifikasi Kanker Payudara.

Model

Menggunakan InceptionResNetV2 dengan CBAM (Convolutional Block Attention Module) untuk meningkatkan fokus pada fitur penting.

Preprocessing

Setiap gambar diproses dengan CLAHE (Contrast Limited Adaptive Histogram Equalization) untuk mempertajam detail gambar.

Backend

Dibangun menggunakan Flask (Python) untuk melayani model dan mengelola permintaan unggahan gambar dari pengguna.

Frontend

Desain antarmuka (UI/UX) yang bersih dan responsif dibangun menggunakan Bootstrap 5 dan file CSS/JS eksternal.
